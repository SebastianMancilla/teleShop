# Teslo API

1. Clone the repo

2. Execute the command:
```
  yarn install
```

3. Clone file ```.env.template``` and rename as ```.env```

4. Change the env variables

5. Up the BD with
```
  docker-compose up -d
```

6. Up the proyect ```yarn start:dev```
